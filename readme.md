## 微信-图灵机器人

#### 背景

* [vbot](https://github.com/HanSon/vbot)
* [图灵API](http://www.tuling123.com/)

#### 安装

```
git clone https://github.com/yaoshanliang/vbot-tuling.git
cd vbot-tuling
composer install
```

#### 运行

```
php run.php
```

开始关键字:**聊天**

退出关键字:**不聊了**
